- np.array([1,2,3,4], dtype='int8'|'int32'|float)

```python
test4 = np.array([[1, 2, 3, 4]], dtype = 'int8')
print(test4.dtype)
print(test4.itemsize)   #전체 원소 개수
print(test4.ndim)       #차원
print(test4.size)       #전체 바이트 용량
print(test4.shape)      #구조 - 튜플(4, ) (1,)
print(test4.strides)
```
